About DoD Budget Explorer

DoD Budget Explorer is an independent research tool for exploring U.S. Department of Defense budget justification data. It provides full-text search, interactive visualizations, and program-level analysis across fiscal years, military services, and appropriation accounts.

Data Sources

All data is extracted from publicly posted budget justification documents published by the DoD Comptroller and individual service budget offices (Army, Navy, Air Force, Space Force, Defense-Wide agencies). The database covers 9 exhibit types including P-1, P-5, R-1, R-2, O-1, M-1, C-1, and more.

How to Use

  • Dashboard — High-level overview of budget totals, service breakdowns, year-over-year trends, and top programs.
  • Search — Filter budget line items by keyword, fiscal year, service, exhibit type, appropriation, and amount range. Download results as CSV, JSON, or Excel.
  • Charts — Interactive visualizations including treemaps, service comparisons, appropriation breakdowns, and year-over-year delta analysis.
  • Program Explorer — Deep-dive into individual Program Elements (PEs) across all fiscal years, exhibits, and narrative descriptions from budget justification PDFs.

API Access

Programmatic access to all budget data is available via the REST API. Endpoints support filtering, pagination, full-text search, aggregations, and data export in CSV, JSON, and Excel formats.

Methodology

Budget documents are downloaded from official DoD sources, parsed from Excel and PDF formats, and loaded into a SQLite database with full-text search indices. Monetary values are normalized to thousands of dollars ($K). Data validation includes cross-service reconciliation, cross-exhibit consistency checks, and year-over-year outlier detection. See the API documentation for complete details.
