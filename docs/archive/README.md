# Archived Documentation

This directory contains historical development documents that are no longer actively maintained but are preserved for reference. These include agent instruction files, implementation logs, planning documents, and optimization analysis from the project's development phases.

## Contents

| Directory | Description |
|-----------|-------------|
| `instructions/` | Agent instruction files (LION, TIGER, BEAR, OH MY) used during parallel development |
| `implementation-logs/` | Implementation summaries, fix logs, and status documents |
| `planning/` | TODO breakdowns, wireframes, GUI plans, and task indices |
| `optimizations/` | Detailed optimization analysis and testing documentation (27 files) |
